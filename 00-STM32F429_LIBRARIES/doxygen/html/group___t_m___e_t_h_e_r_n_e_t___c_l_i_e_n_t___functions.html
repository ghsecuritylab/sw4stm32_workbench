Makes TCP CLIENT Connection to external server.

Note
After this call, different client callbacks will be called depending on what will happen.
Parameters
*conn_nameConnection name. This value will be then passed to all client based handlers
ip1IP address. ip1 is MSB, ip4 is LSB
ip2IP address. ip1 is MSB, ip4 is LSB
ip3IP address. ip1 is MSB, ip4 is LSB
ip4IP address. ip1 is MSB, ip4 is LSB
portPort to which we want connect to server
*user_parametersPointer to user data to be passed to this function. This data will be used in all client based callbacks inside TM_TCPCLIENT_t pointer variable Use NULL in case you don't want to pass any data.
Return values
